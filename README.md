### 二、运行

##### 1.启动服务器 （人人对弈时需要）
切换到 `build/` 目录下，执行 `java -jar gobang-server.jar` 即可。

##### 2.启动客户端
切换到 `build/` 目录下，双击 `gobang.jar` 即可。
